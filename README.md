## 📖 About the Project

This project is a **Digital Construction Management System** designed to optimize the workflow of civil engineering teams. It replaces traditional paper-based reports with a responsive web application, allowing real-time tracking of:

1.  **Studios Delivery Checklist:** Tracking finishing details (painting, hydraulics, electrical) for multiple units.
2.  **R.D.O (Daily Construction Report):** A digital version of the official engineering daily log, featuring workforce tracking and climate monitoring.

The system was built with a focus on **Mobile First** design to ensure usability directly at the construction site via smartphones.

---

## 🚀 Features

* **Real-time Search:** Filter items by name, function, or status instantly.
* **Responsive Design:** Optimized layout for Mobile, Tablet, and Desktop.
* **Engineering Standards:** Visual style mimics official engineering documents (A4 format, technical headers).
* **Print-Ready:** Specialized CSS (`@media print`) to generate clean PDF reports for physical signing.
* **Visual Status Indicators:** Automatic color coding for "OK", "Pending", and "In Progress".

## 📱 How to Use

1.  **Access the Live Version:** [Click here to view the Dashboard](https://marcossoftwareengineering.github.io/Studios/)
2.  **Search:** Use the input bar at the top to filter specific construction items or workers.
3.  **Mobile View:** Open on your smartphone to see the responsive table adaptation (horizontal scroll).
4.  **Print Report:** On the Desktop version, press `Ctrl + P` (or Cmd + P) to generate the formatted A4 PDF.
